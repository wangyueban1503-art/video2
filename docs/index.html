把你的视频文件命名为 video.mp4 并上传到 docs/videos/，然后访问此页面播放。

上传视频（简易指南）

  1. 在本地把你的视频文件重命名为 video.mp4（或同时放入其他格式），然后把它放到 docs/videos/。
  2. 通过 GitHub 网页界面上传：进入仓库 -> 点击 "Add file" -> "Upload files"，把文件拖到 docs/videos/ 路径并 Commit。
  3. 或用 git 命令行： 
    git clone https://github.com/wangyueban1503-art/video2.git
cd video2
mkdir -p docs/videos
cp /path/to/your/video.mp4 docs/videos/video.mp4
git add docs/videos/video.mp4
git commit -m "Add video file"
git push

注意：GitHub 单文件上传限制约为 100MB；若视频较大（>100MB），请参照 docs/README.md 中的替代托管方案。
